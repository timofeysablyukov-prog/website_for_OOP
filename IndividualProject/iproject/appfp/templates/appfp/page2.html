Создание классов и объектов

Создание классов

Класс - это создаваемый разработчиком тип данных. Создавая класс, мы создаём свой тип данных и получаем возможность создавать объекты принадлежащие к данному классу.

В языке python класс создаётся при помощи ключевого слова class, после которого через пробел идёт само название класса. На следующей строке уже в самом классе идёт метод init который задаёт поля (атрибуты класса). Поля, задаваемые вместе с созданием класса определяются при помощи слова self (self.*название поля* = *название поля*). Пример:

По-простому, класс - это шаблон для создания вещей по данному шаблону.

class Example:
    def __init__(self, field1, field2):
        self.field1 = field1
        self.field2 = field2
        self.field3 = 0  # поле со значением по умолчанию

Создание и вызов методов класса

Как вы поняли с прошлого урока методы - это действия наших с вами объектов. Тут всё то же самое как с функциями только внутри класса и первый аргумент всегда self (пока что будем использовать только такой). Обычно методы используются для взаимодействия со значениями полей класса.

Пример класса с использованием метода:

class Car:
    def __init__(self, name):
        self.name = name
        self.place = [0, 0]

        
    def drive(self, x, y):

        
        self.place[0] += x

        
        self.place[1] += y

    

    
Данный пример описывает примитивную машину, которая из данных имеет только имя и место (координаты). А также она имеет функцию езды, которая представляет собой изменение местоположения.

    
Здесь слово self обозначает обращение к объекту класса и к его полю (после точки).

    
Создание объектов классов

    
Объект - это конкретный экземпляр класса. То есть вещь которую мы создали. Это вещь в коде, которую мы создали по шаблону(классу). Объект создаётся следующим образом:

    

        
# Создание объекта

        
my_car = Car("Toyota") # название объекта = класс и поля (в скобках)

        
# Вызов метода объекта

        
my_car.drive(10, 5)

        
# Доступ к полям объекта

        
print(my_car.name)  # Вывод: Toyota

        
print(my_car.place) # Вывод: [10, 5]

    

    
После создания объекта мы можем работать с его методами и полями, используя точечную нотацию (объект.метод).
