Функции и методы

Понятие функции

Функции - это действия программы и её объектов. Функции могут принимать аргументы - переменные или объекты, с которыми может взаимодействовать функция.

Примеры функций:

   print(*аргументы*) - выводит на консоль значение аргумента. Может принимать несколько аргументов.
   sum(*аргумент*) - выводит сумму элементов массива. Может принимать только один аргумент.
   input() - выводит значение(строка), вводимое пользователем с клавиатуры на консоль. Не принимает аргументов.
   str(*арг*), int(*арг*), float(*арг*) - выводят аргумент с изменённым типом данных(зависит от функции). 1 арг.

Вы наверное уже заметили, что функция print выводит данные на консоль, а остальные выводят данные непонятно куда. Они выводят данные туда, куда им скажет разработчик. То есть их можно как записать в переменную, так и взаимодействовать с ними при помощи других функций, например вывести данные на консоль при помощи функции print.

Создание функции

Python, как и многие языки программирования предоставляет возможность создавать нам свои функции. Как создавать функции:

def *название функции*(*переменная(ые)-аргумент(ы)*):
   *тело функции(код)*
   return *вывод функции(данные)*

Пример:

def example(arg):
   result = arg+1
   return result

Данная функция выводит значение переменной result, которое на 1 больше значения переменной arg, являющейся аргументом.

Вызов функции

Чтобы вызвать функцию, нужно написать её название, а после него в скобках аргумент(ы). В коде это выглядит так

func_name(args)

Понятие метода

Функция, распространяющаяся только на объекты классов или переменные типов данных, называентся методом. Существуют встроенные методы для уже существующих типов данных. Их вызов осуществляется следующим способом:

object.method(arg)

Сначала идёт объект, а затем через точку метод и в скобках аргумент(ы)

Методы можно создавать только для своих классов. Об их создании поговорим позже.
