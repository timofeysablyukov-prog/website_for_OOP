Абстракция

Что такое абстракция?

Абстракция - это принцип ООП, который позволяет скрыть сложные детали реализации и показать только необходимые характеристики объекта. Мы работаем с объектами на высоком уровне, не вдаваясь в детали их внутреннего устройства.

Пример из реальной жизни

Когда вы управляете автомобилем, вам не нужно знать как работает двигатель, система зажигания или трансмиссия. Вы используете руль, педали и рычаг передач - это интерфейс, абстрагирующий сложность автомобиля.

Абстракция в Python

В Python абстракция достигается через создание классов, которые определяют только существенные характеристики объекта. Мы скрываем детали реализации методов.

class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self.__balance = balance  # скрытый атрибут
    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount
            return True
        return False
    def withdraw(self, amount):
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            return True
        return False
    def get_balance(self):
        return self.__balance

В этом примере пользователь не знает как именно хранится баланс (это могут быть рубли, доллары, копейки), он просто использует методы deposit(), withdraw() и get_balance().
