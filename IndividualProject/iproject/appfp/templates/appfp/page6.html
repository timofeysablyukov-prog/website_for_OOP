Полиморфизм

Что такое полиморфизм?

Полиморфизм (от греч. "много форм") - это возможность объектов с одинаковым интерфейсом иметь разную реализацию. Один и тот же метод может работать по-разному для разных классов.

Полиморфизм в действии

class Bird:
    def move(self):
        return "Летит"
class Fish:
    def move(self):
        return "Плывёт"
class Emu:
    def move(self):
        return "Бежит"
# Все объекты имеют метод move(), но реализация разная
animals = [Bird(), Fish(), Emu()]
for obj in animals:
    print(obj.move())

Вывод программы:

Летит
Плывёт
Бежит

Полиморфизм и утиная типизация

В Python используется "утиная типизация" (duck typing): если объект ходит как утка и крякает как утка, то это утка. Важен не тип объекта, а наличие нужных методов.

class RussianSpeaker:
    def speak(self):
        return "Привет!"
class EnglishSpeaker:
    def speak(self):
        return "Hello!"
class Dog:
    def speak(self):
        return "Гав!"
def greet(speaker):
    print(speaker.speak())
# Функция greet работает с любым объектом, у которого есть метод speak()
greet(RussianSpeaker())  # Привет!
greet(EnglishSpeaker())  # Hello!
greet(Dog())            # Гав!

Перегрузка операторов

Полиморфизм также проявляется в перегрузке операторов:

class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    def __add__(self, other):  # перегрузка оператора +
        return Vector(self.x + other.x, self.y + other.y)
    def __str__(self):  # перегрузка преобразования в строку
        return f"Vector({self.x}, {self.y})"
v1 = Vector(1, 2)
v2 = Vector(3, 4)
v3 = v1 + v2  # используем перегруженный оператор +
print(v3)      # Vector(4, 6)

Преимущества полиморфизма

1. Гибкость кода

2. Упрощение взаимодействия между объектами

3. Возможность расширения без изменения существующего кода

4. Более читаемый и понятный код

Абстрактные классы

Для создания полиморфных интерфейсов можно использовать абстрактные классы:

from abc import ABC, abstractmethod
class Shape(ABC):
    @abstractmethod
    def area(self):
        pass
class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height
    def area(self):
        return self.width * self.height
